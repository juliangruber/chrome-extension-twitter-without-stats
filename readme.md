
# Twitter Without Numbers

Get a more meaningful twitter experience focused on contents by removing all the numbers!

## Removes

- Tweet count
- Follower count
- Following count
- Likes
- Retweets
- Comment count
- Who To Follow
- Your Tweet Activity
- Trends
